# Hotel Booking EDA Analysis

## Overview

This project is based on Exploratory Data Analysis (EDA) of hotel booking data. The main purpose of this project was to analyze booking trends, customer behavior, cancellation patterns, and hotel revenue insights using Python.

The dataset contains booking information for both City Hotels and Resort Hotels. By analyzing the data, useful business insights were generated that can help hotels improve customer experience and business performance.

---

## Problem Statement

Hotel bookings are influenced by many factors such as season, hotel type, booking channel, cancellation rate, customer type, and demand patterns. Understanding these factors is important for hotel management to make better business decisions.

In this project, the hotel booking dataset was analyzed to identify important patterns and trends through data cleaning, visualization, and exploratory data analysis.

---

## Dataset Information

The dataset contains:
- 119,390 rows
- 32 columns

Some important features in the dataset are:
- Hotel type
- Booking cancellation
- Lead time
- Arrival date
- Customer details
- ADR (Average Daily Rate)
- Market segment
- Distribution channel
- Reservation status

## Key Insights

- City Hotels received more bookings than Resort Hotels.
- Maximum hotel bookings were observed in August.
- Most bookings were done through Travel Agents and Tour Operators (TA/TO).
- Portugal had the highest number of hotel guests.
- City Hotels generated higher revenue compared to Resort Hotels.
- Higher cancellation rates were observed in bookings without deposits.
- Most customers preferred weekday stays.

The project helped in understanding hotel booking behavior and customer trends using real-world data. The insights generated from the analysis can help hotel businesses improve planning, reduce cancellations, and increase revenue.
